# CLAUDE.md

This project inherits all shared conventions from the top-level CLAUDE.md.

## Commit Message Style

```
[component] brief description

Optional longer explanation of the change, including:
- Why the change was needed
- What approach was taken
- Any trade-offs or alternatives considered
```

Examples:
- `[span] add support for byte offset spans`
- `[formatter] fix multi-span rendering alignment`
- `[tests] add property tests for span conversions`
